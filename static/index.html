About Project

This project represent a leaderboard hosting website. It allows authorized users to send the results of their Competitions through our api using an api key. The data will be rendered as a competition leaderboard where you can sort by any chosen column. The technology stack used is pretty straight forward; ExpressJS as the main framework, Postgresql as the database and some CSS magic here and there. This website served as a stepping stone to web development, as I was forced to step out of my confort zone and learn new technologies.
